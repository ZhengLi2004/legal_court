# benchmarks/optim 目录说明

本目录用于参数调优与阈值实验，代码按“数据集构造 -> 阈值扫描/验证 -> 审计复核”组织。

## 目录内容

- `generate_*.py`：构造各类调优样本与评测数据集（如 projection、insight、fact、law、semantic path）。
- `tune_*.py`：执行阈值扫描、训练集/验证集对比并产出最优候选参数。
- `audit_*.py`：对关键参数调整做副作用审计与回归检查。
- `__init__.py`：模块初始化文件。
- `artifacts/`：调优过程产物与大体量中间数据（已在 `.gitignore` 忽略）。

## 使用约定

- 调优脚本优先复用现有运行时逻辑与评分口径，避免离线口径偏移。
- 新增调优任务时，优先补充对应 `generate_*.py` 与 `tune_*.py`，并在必要时增加审计脚本。
- 该目录下的 Python 代码保持原有功能，不在文档调整中改动实现逻辑。

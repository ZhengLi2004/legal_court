# 数据说明

本文件基于 [实验手册.md](/home/lizheng/legal_court/实验手册.md) 与当前仓库内已落盘产物整理，目标是明确：

- 哪些数据是**正式实测**、可作为主实验/主表来源；
- 哪些数据是**当前正文/图像正在使用的展示层**；
- 哪些数据是**补充诊断**；
- 哪些数据是**启发式估计**，不能与正式实测混用。

## 1. 使用原则

### 1.1 正式实测数据

正式实测数据以以下两层为准：

- 数据划分 / Gold / Freeze：
  - `benchmarks/experiments/artifacts/splits/`
  - `benchmarks/experiments/artifacts/gold/`
  - `reports/experiments/20260323_step09a_status_small/`
- 主结果重建包：
  - `reports/experiments/20260401_step14_appendix_current/step14/`

### 1.2 当前正文/图像展示层

当前仓库里供正文直接引用的图像目录是：

- `reports/paper_figures/chapter5_20260403/`

需要特别注意：

- 该目录当前 `figure_manifest.json` 显示 `empirical_gain_mode = on`；
- 因此其中 **Claim 1 图像是经验估计版展示层**，不是纯实测图；
- Claim 2/3/4 图像仍以各自现行数据源为准，具体在下文逐项说明。

### 1.3 启发式估计与补充诊断

这两类数据均有效，但用途不同：

- **补充诊断**：用于 rebuttal / 附录 / 机制解释；
- **启发式估计**：用于当前经验预测展示，不应替代正式实测表。
